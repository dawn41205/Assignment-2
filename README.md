## 使用步驟說明:
1.首先執行input.py<br>
2.透過終端機依次輸入日期、金額、類別、備註<br>
3.輸入完成後，輸入y表示繼續輸入下一筆資料，輸入n表示結束輸入<br>
4.結束輸入後，將資料以下列格式保存至 expenses.json
```
[
    [120, "food"],
    [80, "transport"],
    [200, "shopping"]
]
```
5.然後執行output.py
6.讀取expenses.json
7.根據不同類別的金額加總所佔的百分比，生成圓餅圖並展示於螢幕。<br>
